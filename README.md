# kTYPr_EcoEpidem

This repository contains code and data used for the analysis of *E. coli* transporter-dependent-capsule (K-type) diversity, ecology and epidemiology. This refers to Figures 3-4, S3-S7 and Tables S4-6 in the study "The natural diversity of *E. coli* transporter-dependent capsules".

## Repository structure

```text
code_Ecocapsules.Rmd   # Main analysis and figure-generation code
inputdir/              # Input data
outputdir/             # Processed outputs including supplementary tables
plotdir/	       # Empty directory to reproduce all figures.
